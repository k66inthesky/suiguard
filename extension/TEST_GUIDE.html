🔍 黑名單檢測測試：

1. 點擊「黑名單檢測」頁面
2. 選擇地址類型（Coin Address、Object ID、Domain、Package ID）
3. 輸入測試地址，例如：
0x1234567890abcdef (測試用)
4. 點擊「🔍 檢測黑名單」按鈕
5. 查看檢測結果，系統會顯示該地址是否在 GitHub 黑名單中
